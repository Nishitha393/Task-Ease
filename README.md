<h1><strong>Task-Ease</strong></h1>

<h2><strong>Tech for mobile apps final project</strong></h2>

<p>This React Native app is a simple task management tool called "<strong>Task Ease</strong>." It allows users to log in, sign up, add tasks, edit task completion percentage, and log out. The app's main features include:</p>

<ul>
  <li><strong>Authentication:</strong> Users can either log in with existing credentials or sign up for a new account.</li>
  <li><strong>Task Management:</strong> Once logged in, users can add tasks to their to-do list. Each task can have a completion percentage associated with it.</li>
  <li><strong>Task Editing:</strong> Users can edit the completion percentage of tasks to track their progress.</li>
  <li><strong>Logout:</strong> Users can log out of their account when they're done using the app.</li>
</ul>

<h2><strong>Execution Steps:</strong></h2>

<ol>
  <li><strong>Install dependencies:</strong> Run <code>npm install</code> or <code>yarn install</code>.</li>
  <li><strong>Start the Metro Bundler:</strong> Run <code>npx react-native start</code>.</li>
  <li><strong>Run the app on iOS simulator:</strong> Run <code>npx react-native run-ios</code>.</li>
  <li>The app will launch on the iOS simulator, where you can interact with it to manage tasks.</li>
</ol>
